Mandrill (now officially called Mailchimp Transactional Email) API key works as an email provider. It is designed specifically for sending automated, 1-to-1 transactional emails (such as order confirmations, password resets, and welcome emails) rather than bulk marketing newsletters. 


How it Works
The Mechanism: Your Mandrill API key serves as your authentication credential. You use it to connect your app or website to Mandrill's servers via their REST API or SMTP Relay. 

Delivery: Once your app passes the API key along with the email content, Mandrill routes and delivers the email to the recipient's inbox.
